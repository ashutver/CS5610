Introduction

Fitness Plus is my project under the course Web Developemnt (CS-5610) that I took during my graduation. I have created a applciation on fitness. User can look at different exercises and create his workout. Basically people can find all information related to fitness whether it is about muscles or equipment or nutrition.
I have used following api for my project: Wger api

List of available functionalities :

Functionality Description
Workouts user can create/edit/view/delete his workouts. user can view published workouts by other users.
Workout follow/unfollow User can follow/unfollow workouts of other users
Exercise Exercise page displays all exercises grouped in categories.
Exercise Details Details page displays exercise details.
Exercise Details Comments User can view all comments on exercise details page. To write a comment user has to login. User can view profile of users who commented here
Muscle Major Body muscles are shown and related exercises are shown
Equipment Equipments are shown here and once user clicks on them related exercises will be shown
Nutrition Page displays different food items. User can search for a food item.
Nutrition Details Details page displays nutrition details for a food item

Different Pages of Project :

PAGE DESCRIPTION

Home Page Home Page where only pictures are displayed

Muscle Page User can view different muscles on this page along with the asscociated exercises.

Technologies used:

TECHNOLOGY PURPOSE USED IN
HTML All the pages are designed using HTML Client Side
CSS / Bootstarp This is use to enhance the look and feel of project and bootstarp also makes it responsive Client Side
jQuery It is used to show autocomplete feature along with categories. Client Side
Java Script JS is used to handle some onclick functionalities Client Side
jQuery-UI Used to display popups having autocomplete functionality. Client Side
Angular.JS It is used to handle data from webservices Client Side
Node.js This technology is the backbone of server side functionality. Server Side
Express (Node.js Module) This helps in establishing web services on which the project work. Server Side
MongoDB This is the backend used for data storage purpose Data Storage
Mongoose (Node module) Helps in integrating MongoDB at server side Server Side
