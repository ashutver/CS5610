Introduction


Fitness Plus is my project under the course Web Developemnt (CS-5610) that I took during my graduation. I have created a applciation on fitness. User can look at different exercises and create his workout. Basically people can find all information related to fitness whether it is about muscles or equipment or nutrition.

List of available functionalities :

Functionality Description
Workouts It can be added multiple times and user can configure each of them for different locations
Exercise A user can multiple sticky notes on dashboard to make quick notes
Muscle A user can Google information about some text
Equipment This widget helps in translating text to speech
Nutrition This widgets show time of different time zones depending upon user's choice

Different Pages of Project :

PAGE DESCRIPTION
Login Pop up User can login here. Most of the functionality is avilable to user. Only if user has to create a workout or enter comments at exercise details page then user needs to login
Home Page Home Page where only pictures are displayed
User Profile User profile where user can view his details. User can view the workouts created along with workouts which user is following. user can also view the comments he gave on different exercises. user can view other people profile.
Workout user can view his workouts along with workouts he is following along with published workouts by other users.
Workout Details A user can edit a workout here. User can add more days or exercises to his workout. User can view other people workouts here.
Exercise Page User can view different categories of body parts along with exercises to train them
Exercise Details Page User can view details of an exercise and can also comment on them
Muscle Page User can view different muscles on this page along with the asscociated exercises.
Equipment Page User can view different equipments on this page. User can click on any equipment to view associated exercises.
Nutrition page User can view different food here. User can search too for food here
Nutrition Details page Shows nutrition details for the food item selected

Technologies used:

TECHNOLOGY PURPOSE USED IN
HTML All the pages are designed using HTML Client Side
CSS / Bootstarp This is use to enhance the look and feel of project and bootstarp also makes it responsive Client Side
jQuery It is used to show autocomplete feature along with categories. Client Side
Java Script JS is used to handle some onclick functionalities Client Side
jQuery-UI Used to display popups having autocomplete functionality. Client Side
Angular.JS It is used to handle data from webservices Client Side
Node.js This technology is the backbone of server side functionality. Server Side
Express (Node.js Module) This helps in establishing web services on which the project work. Server Side
MongoDB This is the backend used for data storage purpose Data Storage
MongoJS (Node module) Helps in integrating MongoDB at server side Server Side
Node Mailer (Node Module) Helps in sending automated mails Server Side
