Exercise Functionality

Exercise page groups exercises according to muscle categories. user can view the tab having following categories::

  • Arms
  • Legs
  • Chest
  • Abs
  • Back
  • Shoulder
  • Calves
All exercises are displayed as a link and user can click on them to move to Exercise Details page
