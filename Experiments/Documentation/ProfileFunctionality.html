Profile Functionality

Profile page shows the user details. It also shows the workouts that user has published and the workouts that the user is following. It also displays the user comments that user has written on exercise detail pages. Following things are shown::

  • User details
  • Followers
  • Following
  • Comments


Snapshot

Profile Services

Profile services are designed using Node.js and Express Module. MongoDB is used for storing all the user data. Following services are are playoing role on login page:

  1. Fetches user details
  2. Fetches user workouts that user has published
  3. Fetches workouts whom user is following
  4. Fetches user comments on exercises
