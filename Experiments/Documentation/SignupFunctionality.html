Signup Functionality

SignUp page where user can sign up. Following fields are to filled

  • Username
  • FirstName
  • LastName
  • Email
  • Password
Username is a unique field. I am handling this at client side where I check the username entered with all usernames present in database and will only allow the control to hit nodeJS if username does not exists in database. Once user signup then user is shown profile page by default.
