Workout Functionality

Workout page shows details about workouts.

  • User can create/edit a workout here
  • User can publish his workout so that other users can view it and can follow it.
  • All published workouts are shown to user
  • User can follow/unfollow a workout
  • User can view all followed workouts


Snapshot

Workout Services

Workout services are designed using Node.js and Express Module. MongoDB is used for storing all the user data. Following services are are playoing role on login page:

  1. Fetch workouts created by user.
  2. Fetch published workouts which the current user has not followed.
  3. Fetch workouts that user is following.
  4. Create/edit a workout.
  5. Follow/unfollow a workout.
  6. Edit/Delete a workout name.
