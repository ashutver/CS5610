Using CSS for styling

This is an example of inline CSS. An inline style loses many of the advantages of a style sheet (by mixing content with presentation). Use this method sparingly! To use inline styles, add the style attribute to the relevant tag. The style attribute can contain any CSS property.


Leftmost column. Here background color is aqua

Center column. Here background color is slategray

Rightmost column. Here background color is salmon


Text is aligned in center. Here one div is inside another

About the experiment

This experiment demonstrates how inline css can be replaced by using a stylesheet. Top portion shows 3 div block placed adjacent to each other. Below we can see one div block placed at center of another div

Code Snippet

            .left{
                  background-color: aqua; width: 250px; height: 250px; float:left
                 }
            .center{
                   background-color: slategray; margin: auto;position:absolute; left: 0; right: 0; 
                   width: 250px; height: 250px; float: left; 
                   }
            .right{
                   background-color: salmon; width: 250px; height: 250px; float: right;
                 }
