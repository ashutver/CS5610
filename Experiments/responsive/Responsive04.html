Responsive Image

About the experiment

This Experiment demonstrates responsive image. Here the image adjusts to the given screens like dektops, tablets or mobile phones. As the width of the screen decreases the image is resized to fit in the window.

Code Snippet

             @media screen and (min-width:540px) and (max-width:600px) {
             body {
                   background-color: darkslateblue;
                  }
             #image-res {
                    width:530px;
                   margin: auto;
                        }
                   }
