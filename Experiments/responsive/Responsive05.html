About the experiment

Extending Responsive Experiments, I have worked on a table. In bigger screens table can be well placed but in samaller screens it is very difficult to present a table in tabular form so for smaller screens, using CSS, I am showing data in form of sheets.

Code Snippet

             @media screen and  (max-width:500px) {
             body {
                     background-color: aliceblue;
              }
             table,thead,tbody,th,td,tr{
                display:block;
               }
             table{
               width:100%
              }
             thead tr {
                    position: absolute;
                    /*here we need to hide the headers*/
                    top: -9999px;
                    left: -9999px;
                }
             tr {
               border: 1px solid indianred;
               }
              td {
                /* td should behave as row */
                border: none;
                border-bottom: 1px solid cadetblue;
                position: relative;
                padding-left: 65%;
               }
             td:before {
                    /*row header */
                    position: absolute; 
                    top: 6px;
                    left: 6px;
                    width: 45%;
                }
             /*Labeling the data*/
                  td:nth-of-type(1):before {
                    content: "Book Name";
                   }
                  td:nth-of-type(2):before {
                    content: "Book Author";
                  }
                  td:nth-of-type(3):before {
                    content: "Book Cost";
                  }
                  td:nth-of-type(4):before {
                    content: "Availability";
                  }
         
              }
