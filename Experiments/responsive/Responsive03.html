About the experiment

Extending Experiment1, here I added a menu bar stretched horizontally for wider screens like desktop and tablets, but for smaller screens like mobile screens, this tabular form is replaced by a single menu. once you click on it it takes user to bottom where the links are displaced horizontally.

Code Snippet

           #menu-link,#small-menu{
            display:none;
            }
             @media screen and (max-width:500px) {
              #menu-link,#small-menu{
                display:block;
            }
        }
