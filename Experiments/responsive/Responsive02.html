About the experiment

With a @media query, you can write different CSS code for different media types. This helps when you want different layout for different media types such as a screen or a printer, but also when you want different layout for different devices, which is very useful when making web pages with responsive design. You can also have different layout when a user resizes the browser window up or down to a certain width, or height. Here I used 3 media tags where first one is for desktop, where 2 section are placed horizontally. Next one is used for tablet, here also sections are arranged horizontally. Next one is for mobile. here input label and textbox will be placed vertically and sections will also be placed vertically.

Code Snippet

           @media screen and (max-width:500px) {
            body {
                background-color: orangered;
            }
            @media screen and (min-width:500px) and (max-width:700px) {
            body {
                background-color: moccasin;
            }
