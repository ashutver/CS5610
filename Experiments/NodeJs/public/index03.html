Add data using NodeJS

About the experiment

This experiment demonstrates the Node framework. Here I am using Node frameqwork to add static data. Intially I have created a table on UI that uses angular to call a service. this service will be created in Node server and exposed. Using this service angular will collect data and populate the table on UI This is a simple experiment demonstrating the use of NodeJS and user can input the data in given text box and then can add a new record.

Code Snippet

                    Client side  >>
                      app.controller("DeveloperController",
                        function ($scope,$http) {
                            $scope.edit = null;
                            $http.get("/api/developer")
                            .success(function (response) {
                                $scope.developers = response;
                            });
                     $scope.add = function (developer) {
                      $http.post("/api/developer", developer)
                       .success(function (response) {
                        $scope.developers = response;
                       });
                     }
                  Server Side >>
                       app.get("/api/developer", function (req, res) {
                        res.json(developers);
                       });
                       app.post("/api/developer", function (req, res) {
                        var obj = req.body;
                        developers.push(obj);
                        res.json(developers);
                     });

Code Snippet Explanation

Here at client side I am using a controller and initally I will get all the records from the service. I will create the service that will return the static data and this data will be collected by angular to show in table format. here user can add a record. In Angular I will call Node serveice using post and data will be sent to server where it will be added to given array.
