Display data based on a drop down

About the experiment

This experiment demonstrates how an AJAX can be used to fetch data from the server and render that data on your UI. The web service used for this experiment is wger Workout Manager API and data type received is JSON. This is a simple experiment demonstrating the use of AJAX and a dropdown is shown and for every category a value is assined. This value will be appended to api call as aparameter and data will be displayed


  • Title of Exercise will be displayed here

    Description of Exercise will be displayed here

Code Snippet

                                    <select class="exercise">
                                    <option>Choose a category </option>
                                    <option value="10">Abs </option>
                                    <option value="8">Arms </option>
                                    <option value="9">Legs </option>
                                    <option value="13">Shoulders </option>
                                </select>
                              function renderExercises(exercises) {
   
                                 var parent = $("#Exercises");
                                 var template = parent.find(".template");
                                 parent.empty();
                                 var results =exercises.results; 
                                 for (var m in results) {
                                    var exer = results[m];
                                    var title = exer.name;
                                    var desc = exer.description;
       
                                    var item = template.clone();
                                    item.find(".title").html(title);
                                    item.find(".plot").html(desc);
                                    parent.append(item);
       
                                    var id = exer.id;
       
    }
}

Code Snippet Explanation

Here basically a AJAX call is made to fetch data from a web service and a dropdown is shown and for every category a value is assigned. This value will be appended to api call as aparameter and data will be displayed. There is a template on ui, I will grab that template and empty its contents. Then I will add my data to this template and show it on UI.
