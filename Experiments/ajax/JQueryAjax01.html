Fetching data using AJAX

About the experiment

This experiment demonstrates how an AJAX can be used to fetch data from the server and render that data on your UI. The web service used for this experiment is wger Workout Manager API and data type received is JSON. This is a simple experiment demonstrating the use of AJAX and required data is displayed on the screen

    Code Snippet

                $(function () {
                 $.ajax({
                      url: "https://wger.de/api/v2/exercise/?format=json&language=2&limit=5",
                      success: renderExercises
                      });
              });
               function renderExercises(exercises) {
                     var ul = $("#Exercises");
                     ul.empty();
                     var results =exercises.results; 
                     for (var m in results) {
                     var exer = results[m];
                     var title = exer.name;
                     var desc = exer.description;
                     var p = $("<p>").append(desc);
                     var li = $("<li>").append(title)
                           . append(p);
                     ul.append(li);
                     }
                  }

    Code Snippet Explanation

    Here basically a AJAX call is made to fetch data from a web service and then I am parsing over the data received in results. Then I will map my fields from the data received and I am creating line item components dynamically and adding rows to it and at end I will add the rows to the unordered list
