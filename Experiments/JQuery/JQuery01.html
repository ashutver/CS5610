JQuery Experiments

About the experiment

This experiment shows the effect achieved using JQuery.

Here you can drag any of the below screens.

Indian

American

Chinese

Italian

Code Snippet

            $(function () {
                  $(".Indian").draggable({
                    grid:[20,20]});
                  $(".American").draggable({
                    grid: [20, 20]});
                  $(".Chinese").draggable({
                     grid: [20, 20]  });
                  $(".Italian").draggable({
                  grid: [20, 20]  });
                })

Code Snippet Explanation

Here I am capturing the JQuery object and making it draggable. Moreover I am assigning draggable with a additional property of having grid(20,20), means it can be dragged by grid size 20,20.
