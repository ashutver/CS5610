Populate data using Angular

About the experiment

This experiment demonstrates the Angular framework. Here I am using Angular frameqwork to fetch data from the web API. I will assign the data to scope variables and it render on your UI using the placeholders. This is a simple experiment demonstrating the use of Angular and required data is displayed on the screen

Code Snippet

                          app.controller("ExerciseController",
                            function ($scope,$http) {
       
                                $http.get("http://wger.de/api/v2/exercise/?format=json&language=2&&limit=5&callback=JSON_CALLBACK")
                                .success(function (response){
                                    $scope.exercises = response.results;
                            });
                              $scope.removeExercise = function (exercise) {
                                    var index = $scope.exercises.indexOf(exercise);
                                    $scope.exercises.splice(index, 1);
                                }
                           <tr ng-repeat="exercise in exercises">
                           <td>exercise.title</td>

Code Snippet Explanation

Here I am using Angular frameqwork to fetch data from the web API. Above mentioned is the way to achieve this. After recieving the data I am assigning the results to a scope variable exercises. As there we will be placeholders on my UI to collect this data so it will be rendered on UI. If I make any change to data in model then it will be reflected on UI.
